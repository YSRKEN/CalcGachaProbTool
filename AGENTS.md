# AGENTS 運用ルール

このリポジトリでは、次の運用ルールを適用します。

## ドキュメント更新ルール

- 実装や設定を変更した場合は、同一PR内で `docs/` も同時に更新すること。
- docs 更新が不要な場合は、PR本文に「docs更新不要」の理由を明記すること。

## ブランチ運用ルール

- 基準ブランチは `master` とし、作業は `codex/*` 形式の作業ブランチで行うこと。
- コミットは「1目的1コミット」で小さく分割すること。

## ローカル管理ディレクトリ

- `.serena/` はローカル補助情報のため、Git 管理対象に含めないこと。
